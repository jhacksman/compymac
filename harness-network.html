5. Data Flow Constraints

Truncation Points

  • File reads - partial view threshold
  • Shell output - ~30,000 chars
  • Browser HTML - always stripped
  • Full content saved to /home/ubuntu/full_outputs/

Authentication

  • Git: Proxy URL (git-manager.devin.ai)
  • APIs: Environment variables
  • Secrets: interpolation only

State Persistence

  • Bash: Per bash_id session
  • Browser: Per tab_idx
  • LLM Context: Session-scoped
